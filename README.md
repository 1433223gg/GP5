# git操作

## 初始化版本库
- git init
- 初始化成功后，当前目录后面有(master)
- 初始化成功后，当前目录下有个隐藏文件.git（不要对其进行任何操作）

## 工作区
- 持有实际文件（当前能直接看到的文件夹文件）
- 我们平时增删改查的文件都是工作区的内容

## 提交到暂存区
- 可以理解为我们看不到的一个地方，不需要直接去操作这个区域
- 本地仓库的一个主要组成部分

## 本地仓库
- 可以理解为我们看不到的一个地方，不需要直接去操作这个区域
- 也是存在于用户电脑中的
- 用于存储项目代码及版本项目记录等信息，一般工作区的文件代码都在本地仓库中

## 提交到暂存区
- git add 文件名
- 将工作区的变动提交到暂存区（增删改）
- git add .
- 将当前工作区目录下的所有变动提交到暂存区

## 查看工作区和暂存区的状态
- git status
